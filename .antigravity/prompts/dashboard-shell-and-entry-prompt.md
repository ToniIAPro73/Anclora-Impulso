# Prompt: Dashboard Shell And Entry

Implementa features de shell privado y entry flow en Anclora Impulso siguiendo estas reglas:

- `/` debe resolver hacia la entrada principal definida por producto.
- Las pantallas de marketing o landing deben permanecer accesibles por ruta explícita.
- El shell privado debe centralizar header, sidebar, toggles globales y menú de usuario.
- Idioma y tema deben reutilizar contextos globales ya existentes.
- Toda nueva decisión debe quedar reflejada en `sdd/features/dashboard-shell-and-entry`.
