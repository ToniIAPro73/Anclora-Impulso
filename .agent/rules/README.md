# Agent Rules

Este directorio contiene reglas operativas por feature.

Regla mínima:
- Cada feature nueva debe registrar alcance, restricciones y criterios de aceptación.
- La regla debe apuntar a sus artefactos en `sdd/features/...`.
- Toda feature nueva debe cumplir también la regla general de localización y el contrato `LOCALIZATION_CONTRACT.md`.
