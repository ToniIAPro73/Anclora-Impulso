# Agent Rules

Este directorio contiene reglas operativas del proyecto.

Regla minima:
- Cada feature nueva debe registrar alcance, restricciones y criterios de aceptacion.
- La regla debe apuntar a sus artefactos en `sdd/features/...`.
- Toda feature nueva debe cumplir tambien la regla general de localizacion y el contrato `LOCALIZATION_CONTRACT.md`.
