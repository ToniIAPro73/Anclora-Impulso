# Documentacion de Anclora Impulso

Este directorio concentra toda la documentacion no esencial para el arranque del repositorio.

## Mapa

- `internal/`: documentacion interna de trabajo y contexto para asistentes
- `operations/`: despliegue, operacion y runbooks
- `development/`: integracion, referencias tecnicas y roadmap
- `standards/`: contratos y convenciones internas
- `business/`: material ejecutivo y de negocio
- `investors/`: materiales para inversores
- `archive/`: entregables y documentos historicos
