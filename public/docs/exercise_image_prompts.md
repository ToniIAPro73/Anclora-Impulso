## Instrucciones de Uso

Cada prompt está diseñado para mantener la **consistencia visual** del personaje:
- **Pelo**: Castaño corto
- **Ojos**: Verde vivo
- **Bigote**: Sí
- **Ropa**: Camiseta negra, pantalón corto negro, zapatillas negras

Cada imagen incluye dos poses (START y END) con etiquetas y una flecha de movimiento.
